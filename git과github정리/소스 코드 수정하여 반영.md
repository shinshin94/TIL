# 소스 코드 수정하여 반영

## 해당 프로젝트에 소속된 사람이 아닌 경우

> * 특정한 커뮤니티의 구성원이 아니라서 스스로 커밋을 하여 저장소에 적용할 권한이 없을시 제약이 있음
> * PR(Pull Request)를 작성하여 오픈 소스에 기여할수 있음
> * 해당 오픈 오픈 소스 관리자가 이를 허용했을 때 반영

##해당 프로젝트에 소속된 사람인 경우

> * 이전에 배운 add + commit + push로 반영
>
> 1. git status
>    * 현재 폴더의 상태를 알려줍니다.
>    * 변화가 있는 파일이나 추가된 파일이 있는경우 표시 됩니다.
>    * 변화된 파일이나 추가된 파일을 add나 checkout --으로 선택할수 있음
>      * git checkout -- <변경 파일> 
>        * 해당 파일을 변경전으로 되돌림
> 2. git add <추가할 파일>
> 3. git status
>    * git add 로 추가된게 있다면 대기상태로 표시 됩니다.
> 4. git commit -m "메세지"
> 5. git push
