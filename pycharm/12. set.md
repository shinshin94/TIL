# set

## 기본설명과 종류

> 집합 형태의 구조
>
> * 중복을 허용하지 않는다(unhashable type)
>
> * 입력 순서와 출력되는 순서가 다를수 있음
>
> * 인덱스 사용 불가 : 이미 포함되어 있는 요소를 변경할수 없음
>
> * 추가 , 삭제 가능
>
>   * `add() `1개 추가
>   * `update()` 여러개 추가
>
> * 집합 안에 변경 가능한 항목 포함 불가능
>
>   * 리스트는 포함 불가능, 튜플은 포함 가능
>
> * 요소 삭제 가능
>
>   * `remove()` 요소 값이 없는 경우 에러
>   * `discard()` 요소값이 없는 경우 에러 없음
>
> * 요소 전체를 지움
>
>   * `clear()`
